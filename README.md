# Diabetes Prediction Model
### Machine Learning Project
**By: Tiffany Santiago & Ryan Bouapheng**

---

## Overview
This project is a machine learning-based application that predicts the likelihood of diabetes using patient health data. It provides both prediction results and basic interpretability, allowing users to understand how input features influence predictions. All models are implemented in Python within Jupyter Notebooks.

---

## Features
* **Risk Prediction:** Assess diabetes risk based on key health parameters.
* **ML Pipeline:** Full training and preprocessing workflow.
* **Evaluation:** Built-in tracking for accuracy, precision, recall, and F1-score.
* **Reproducibility:** Datasets and notebooks are included for easy replication.

## Model Evaluation

### The training script outputs metrics such as:
| Metric | Description |
| :--- | :--- |
| **Accuracy** | Overall correctness of the model. |
| **Precision** | Accuracy of positive predictions. |
| **Recall** | Ability to identify all actual positive cases. |
| **F1-Score** | Harmonic mean of precision and recall. |
